Check the status of all connected ad platforms and provide a summary of active campaigns.

Use the Synter API to:
1. Check which platforms are connected (Google Ads, Meta, LinkedIn, Reddit, Microsoft Ads)
2. List active campaigns with their current status and spend
3. Flag any platforms that need re-authentication
4. Show a quick health check of campaign performance (any campaigns with anomalous spend or low delivery)

Present the results in a concise dashboard format.
